About Claude Code: The best compliment one can give to it is that it feels like playing a video game.

When you open it for the first time, you think "That's it?" and "Where's the UI?"

It doesn't really tell you how to use it.

But then you start discovering things like:

- Claude.md files
- Slash commands
- Hooks
- Sub-agents
The more you use it, the more interesting it gets, and the more powerful you feel. It's a tool that you can really feel a sense of mastery over.
