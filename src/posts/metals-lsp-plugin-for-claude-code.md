Our latest open-source contribution at Eloquentix: the Metals LSP plugin for Claude Code. This integration brings advanced Scala language server features right into your development workflow. Key highlights include:

- Real-time diagnostics for errors and warnings
- Hover details with types and documentation
- Go-to-definition and find references
- Smart code completions
Ideal for Scala developers aiming to supercharge productivity. Installation is straightforward via Coursier or Homebrew, then simply load it into Claude Code.
