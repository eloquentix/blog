In the agentic world, micromanagement will rule the day. The basics of a good prompt are below, but the more precise your prompt is the better and more consistent your answers will be. As the answers move from agent to agent you have effects similar to tolerance stackup in mechanical engineering.


### 1. Identity and Role

Define who the agent is and its core purpose.


### 2. Scope and Capabilities

Clarify what the agent can do and any special tools or skills it has.


### 3. Behavioral Guidelines

Set the tone, style, and boundaries for responses.


### 4. Task Handling Rules

Define how the agent processes requests and handles edge cases.


### 5. Contextual Awareness

Ground the agent in time, place, or user-specific details.
